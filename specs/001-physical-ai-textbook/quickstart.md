# Quickstart: Physical AI & Humanoid Robotics Textbook

**Feature Branch**: `001-physical-ai-textbook`
**Last Updated**: 2025-12-16

## Prerequisites

Before starting implementation, ensure you have:

- [ ] Node.js 20+ installed
- [ ] Git configured
- [ ] Access to the repository
- [ ] Code editor (VS Code recommended)

## Quick Setup

### 1. Navigate to the book directory

```bash
cd /home/saqib-squad/Hackathon-I/book
```

### 2. Install dependencies (including Mermaid)

```bash
npm install
npm install @docusaurus/theme-mermaid
```

### 3. Start development server

```bash
npm run start
```

The site will be available at `http://localhost:3000`

## Implementation Order

### Phase 1: Configuration (First)

1. Update `docusaurus.config.ts`:
   - Enable Mermaid
   - Set dark mode default
   - Configure GitHub Pages settings
   - Update navbar/footer

2. Update `sidebars.ts`:
   - Configure autogenerated sidebars for modules

3. Update `src/css/custom.css`:
   - Dark mode color variables

### Phase 2: Structure (Second)

1. Create module directories:
   ```bash
   mkdir -p docs/module-01-ros2
   mkdir -p docs/module-02-digital-twin
   ```

2. Create `_category_.json` for each module

3. Create placeholder MDX files for all topics

### Phase 3: Content (Third)

1. Update landing page (`src/pages/index.tsx`)
2. Write Module 1 topics (in order)
3. Write Module 1 assessment
4. Write Module 2 topics (in order)
5. Write Module 2 assessment

### Phase 4: Deployment (Last)

1. Create `.github/workflows/deploy.yml`
2. Test build: `npm run build`
3. Push to trigger deployment

## Common Issues

### Mermaid diagrams not rendering

Ensure both settings in `docusaurus.config.ts`:
```typescript
markdown: { mermaid: true },
themes: ['@docusaurus/theme-mermaid'],
```

### Build fails with TypeScript errors

Run `npm run typecheck` to identify issues before build.

### Dark mode not defaulting

Check `colorMode.defaultMode: 'dark'` in themeConfig.
